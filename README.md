# liveshowdemo
直播app
## How to run
### 1. install RN dependencies
```
yarn install
```

### 2. install iOS SDK
```
cd ios
pod install
```

### 3. run
```
react-native run-ios
```
or

```
react-native run-android
```

react native toast component
